# 终极井字棋 - 游戏说明

## 游戏简介

终极井字棋是一个3x3的大棋盘，每个格子内包含一个3x3的小棋盘，形成9x9的超大棋盘。

## 基础规则

### 落子限制
1. **第一步（先手X）**：可以在任意位置下棋
2. **后续每一步**：玩家只能在**对手上一手落子的位置对应的大棋盘**上下棋
   - 例如：X在位置(大棋盘1, 小棋盘2)落子 → O只能在**大棋盘2**上下棋
3. **自由规则**：如果被锁定的大棋盘已满或已分出胜负，则可以在任意位置下棋

### 获胜条件
- 在大棋盘的9个小棋盘中，获胜次数多的一方获胜
- 小棋盘的获胜条件与普通井字棋相同（三连珠）

## 游戏模式

### 1. 单机模式
- 双人对战，在同一台设备上轮流下棋

### 2. 人机模式
- 与AI对战，三种难度：
  - **简单**：50%概率最优，50%随机
  - **普通**：使用Minimax + Alpha-Beta剪枝算法，追求最优落子
  - **高级**：使用蒙特卡洛树搜索（MCTS）算法（胜率95%+）

### 3. 联机模式
- 双人在线对战
- 创建/加入房间，通过4位房间号匹配
- 房主为玩家X，加入者为玩家O

## 界面说明

- **红色高亮区域**：当前可落子的位置
- **顶部显示**：当前玩家、双方获胜的小棋盘数
- **底部按钮**：重新开始、返回菜单

## 启动方式

### 方式一：PHP内置服务器
```bash
cd 游戏目录
php -S localhost:8000
```
然后访问 http://localhost:8000

### 方式二：其他Web服务器
将文件部署到Apache、Nginx等Web服务器目录下访问

## 文件结构

- `index.html` - 游戏前端
- `server.php` - PHP后端（处理联机对战和高级AI）
- `game_data/` - 游戏数据存储目录（自动创建）
